# CR-0011 SVG Asset Integration

## Summary
방금 정의한 Project K Asset Pack 방향을 실제 SVG 파일 구조로 반영한다.

## Included
- Falcon Lv1~Lv5
- Vulcan Lv1~Lv5
- Aircraft modules
- Enemy role assets
- Boss assets
- Effects
- UI HUD / buttons / icons / frames
- Background / decals

## Note
현재 SVG는 실제 게임 적용 가능한 벡터 에셋 초안이다. 최종 원화 퀄리티 전 단계의 Asset SDK 버전이다.
