# Project K PNG Asset Pack v1.0

승인된 시안 이미지를 기준으로 잘라낸 게임 적용용 PNG 에셋 패키지입니다.

## 포함
- PNG 에셋 83개
- Falcon / Vulcan Lv1~Lv5
- 적 / 보스 / 무기 / FX / UI / 배경
- 게임에 적용된 `index.html`
- `data/assets/asset_manifest.json`
- 미리보기 `docs/preview_assets.html`

## 실행
GitHub Pages에 올리면 실행됩니다.

로컬 테스트:
```bash
python -m http.server 8000
```
