# Project K - Code Handoff v1.5 (UI 디자인 리뉴얼: Sky Force + 미래형)

## 실행
```bash
python -m http.server 8000
```
브라우저에서 `http://localhost:8000` 접속. 9:16 비율 모바일 화면(최대 480px 폭)으로 렌더링됨.

## v1.5에서 추가된 것 — UI/UX 리뉴얼
**톤:** Sky Force 계열 + 미래형 SF UI. 딥스페이스 네이비 베이스 + 시안/일렉트릭 블루 + 마젠타 포인트.

### 디자인 토큰 (style.css :root)
- `--void` #050810 (배경 기본), `--panel` #0c1830 (패널), `--cyan` #4ee3ff (주 강조), `--magenta` #ff4ecf (레벨업/콤보 강조)
- 폰트: Orbitron(디스플레이/숫자), Rajdhani(본문) — Google Fonts CDN, 네트워크 차단 시 시스템 산세리프로 자연스럽게 폴백

### 로비
- 원형 링 프레임 안에 기체가 떠 있는 구조, 점선 링 회전 애니메이션
- 기체 선택을 칩(chip) 형태로, 선택 시 시안 글로우
- 출격 버튼은 육각 컷(clip-path) 처리된 그라디언트 버튼
- 우주 배경 이미지 고정 크기로 상단에 배치, 그 아래는 검은 우주로 자연스럽게 이어짐 (다양한 화면비 대응)

### 인게임 HUD
- 우주 배경이 세로로 무한 루프 스크롤 (seamless tile, 18s 주기)
- HP/EXP 라벨이 화면 밖으로 잘리지 않도록 바 안쪽에 재배치
- 레벨업 버튼 상시 가시성 확보 (비활성 상태도 잘 보이게), 활성 시 마젠타 펄스
- Wave 패널에 STAGE/WAVE를 점(·) 구분자로 한 줄 표시
- 캔버스 단색 배경 제거 → CSS 우주 배경이 그대로 비쳐 보임

### 배경 에셋 처리
- 업로드받은 가로형 우주 이미지를 모바일 폭(480px) 기준으로 가공
- 인게임용은 상하단 페이드 처리로 seamless 세로 반복 타일 제작 (`assets/bg/bg_space_01.png`)
